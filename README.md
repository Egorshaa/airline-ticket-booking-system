Airline Ticket Booking System

Веб-приложение для бронирования и покупки авиабилетов.

Описание

Проект разработан в рамках дипломной работы по специальности 09.02.07 "Информационные системы и программирование".

Система позволяет пользователям регистрироваться, искать рейсы, бронировать билеты и управлять своим профилем.

Администратор может управлять пользователями, рейсами и бронированиями.

Директор следит за учетом.

Оператор слкжбы поддержки имеет возможность отвечать клиентам на вопросы, решать их проблемы.

Возможности
- Регистрация пользователей
- Авторизация
- Личный кабинет
- Поиск авиарейсов
- Бронирование билетов
- Административная панель
- Работа с базой данных MySQL
Используемые технологии
- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
